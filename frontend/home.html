What is GradFlow?

GradFlow is a role-based FYP management system designed for universities to automate document submission, reviews, revision cycles, rubric-based grading, and result compilation.

User Roles

Student

Upload documents, track progress, and view feedback.

Supervisor

Review submissions, approve documents, or request revisions.

Evaluator

Grade projects using standardized rubrics.

FYP Committee

Set deadlines, monitor progress, and release final results.

GradFlow Workflow

Upload Submit Supervisor Review Revision Evaluation Final Result

Key Features

Version Control

Every document upload is logged with timestamps.

Deadline Tracking

Late submissions are clearly marked and notified.

Role-Based Security

Strict access control for transparency and ethics.

Audit Logs

All actions are recorded for accountability.
